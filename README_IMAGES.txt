INSTRUCTIONS FOR IMAGES:

The LaTeX document uses placeholder image names that come with LaTeX:
- example-image.jpg
- example-image-a.jpg
- example-image-b.jpg

These are built-in demo images that should work automatically.

TO USE YOUR OWN IMAGES:
1. Place your image files in the 'images/' folder
2. Supported formats: .jpg, .png, .pdf
3. Replace the placeholder names in sample_document.tex with your actual filenames
   Example: Change "example-image.jpg" to "myimage.png"

IMPORTANT:
- Keep filenames simple (no spaces or special characters)
- Use lowercase
- Good: "figure1.png", "diagram_architecture.jpg"
- Bad: "My Image (1).PNG", "Diagram #2.jpg"

IMAGES FOLDER
=============

Place all your project images here:
- System diagrams
- Architecture diagrams
- Screenshots
- Charts and graphs
- Flowcharts
- UML diagrams

Supported formats: .png, .jpg, .jpeg, .pdf

File naming tips:
- Use lowercase
- Use underscores instead of spaces
- Be descriptive: system_architecture.png, login_screenshot.png
- Avoid special characters
